configuration - 配置文件模板
doc - 使用说明文档
include - 接口头文件
lib - 库文件以及符号文件
tools - 日志解密工具

日志解密工具注意事项：
1. 本工具仅限公司内部使用，严禁对外发布
2. 仅支持hlog 2.2.2或2.2.4+的加密日志解密
3. 仅支持单文件解密，如果有部分加密的字符串跨文件了，那么这部分暂时无法解密
4. 提供两个批处理文件方便对当前目录中的所有日志文件进行整体的解密和解密后的日志的清理，请勿随意更改日志扩展名（不要改成*.p.log扩展名，也不要将*.p.log改成别的扩展名）
5. 如果日志文件处于系统保护的目录，解密工具无法（无权限）读取或生成解密文件，请将原始日志文件拷贝到可执行工具解密的目录进行处理
6. 日志解密工具使用方法详见工具压缩包所携带的说明
